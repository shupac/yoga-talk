Customizing a Sequence

You can customize a sequence by Creating a new sequence or, Editing or Cloning an existing sequence. This will bring you to the Sequence Editor.

The Sequence Editor has two panes: the Sequence Pane (left) and the Details Pane (right). Clicking on an item in the Sequence Pane will allow you to edit that item in the Details Pane.
